# Aqıllı Kitapxana sisteması (Smart Library System)

Bu proyekt Python programmalastırıw tiliniń tiykarǵı túsiniklerin bekkemlew ushın jaratılǵan terminal programma.

### Imkániyatları:
* Kitaplar bazasın kórıw;
* Kitap ijaraǵa alıw (statustı ózgertiw);
* Kitaptı qaytarıw;
* Qátelerdi try-except arqalı saplastırıw.

### Texnologiyalar:
* **Python 3.x**
* **Dictionaries** (Maǵlıwmatlar bazası retinde)
* **While Loops** (Sheksiz menyu ushın)
